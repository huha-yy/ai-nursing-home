## Self-Improving Check

Self-improving skill is at `/opt/openclaw/skills/custom/self-improving/`.

When you encounter:
- **A failed API call** → read `corrections.md`, log what went wrong
- **A user correction** → read `learning.md`, update `memory.md`
- **A repeated mistake** → read `reflections.md`, update `corrections.md`

Quick reference:
- `~/self-improving/memory.md` — active memory (≤100 lines)
- `~/self-improving/corrections.md` — mistake log (last 50)
- `~/self-improving/learning.md` — learning mechanics

The `self-improving` skill is available at
`/opt/openclaw/skills/custom/self-improving/`. Read its SKILL.md and files
when you need to improve from mistakes.
